### To run the project locally:  

### `npm run dev`  

Starts the JSON server at [http://localhost:3010](http://localhost:3010).  

### `npm start`  

Runs the app in the development mode.\  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
